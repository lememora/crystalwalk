Crystal Walk
============

## Setup

Install Node.js:

```
http://nodejs.org/download
```

Clone project:

```
git clone git@bitbucket.org:castilhor/crystalwalk.git
```

```
cd crystalwalk
export PATH=".:$PATH"
```

Install dependencies:

```
npm install
sudo npm install -g bower
bower install
sudo npm install -g grunt-cli
```

Run application:

```
grunt
```
